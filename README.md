## Music Store Data Analysis (SQL Project)

## Project Overview
This project analyzes customer purchasing behavior in a music store database using SQL queries.

## Objectives
- Identify top customers
- Analyze revenue generating cities
- Discover most popular music genres
- Perform business insights using SQL

## Tools Used
- SQL (MySQL)
- MySQL Workbench

## Key Analysis
- Best customer based on spending
- Most popular music genre
- Highest revenue city
- Top artists producing rock music

- SQL queries
